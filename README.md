# studia
Originally stared for a class project, and I have come back and put some extra touches on it. Just a little thought exercise.
The card will show the Italian word, and once clicked, it will show the English translation.
There is an information section that shows you what word youre on, and your prrogress percentage.
There are options to go to the previous or next, as well as restart or shuffle the cards.
The footer contains information for the keyboard shortucts to navigate the game.
